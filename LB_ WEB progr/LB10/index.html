Декоративні CSS-ефекти

Опис ефектів

Цей блок демонструє використання декоративних ефектів CSS: тіней, заокруглених кутів, градієнтів та псевдоелементів.

  • Використання text-shadow для об'єму тексту.
  • Застосування box-shadow для глибини блоків.
  • Створення border-radius для м'яких форм.
  • Налаштування radial-gradient для фону.
