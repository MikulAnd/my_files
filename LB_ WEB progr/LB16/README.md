# Лабораторна робота №16: Основи JavaScript

## Тема
Змінні, типи даних та логічні оператори в JavaScript.

## Виконані завдання
1. **Оголошення змінних:** Створено змінні для зберігання імені, віку та статусу студента.
2. **Перевірка типів:** Використано оператор `typeof` для визначення типів даних (string, number, boolean).
3. **Робота зі складними типами:** - Створено об'єкт `person` (групування даних).
   - Створено масив `hobbies` (список значень).
4. **Логіка та умови:** - Реалізовано перевірку на повноліття.
   - Застосовано оператор нульового злиття `??` для обробки порожніх значень.
5. **Приведення типів:** Продемонстровано конвертацію значень до числа, рядка та логічного типу.

## Як запустити
1. Відкрийте файл `index.html` у будь-якому сучасному браузері.
2. Натисніть `F12` або `Ctrl+Shift+I` та перейдіть у вкладку **Console (Консоль)** для перегляду результатів виконання скрипту.

## Висновок
Під час роботи було опановано базовий синтаксис JavaScript. Засвоєно різницю між примітивними та складними типами даних, а також принципи динамічної типізації в JS.
