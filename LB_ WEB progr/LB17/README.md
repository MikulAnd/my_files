# Лабораторна робота №17: Розгалуження та цикли

## Тема
Алгоритми з розгалуженням та циклічні процеси в JavaScript.

## Виконані завдання
1. **Класифікація даних:** Реалізовано перевірку вікових категорій користувача через `if...else if`.
2. **Обробка варіантів:** Використано конструкцію `switch` для аналізу текстових значень (дні тижня).
3. **Ітераційні процеси:** Виведено послідовність чисел за допомогою циклу `for`.
4. **Вкладені алгоритми:** Побудовано таблицю множення з використанням вкладених циклів.

## Як перевірити
Відкрийте `index.html` та перегляньте вкладку **Console** в інструментах розробника.
