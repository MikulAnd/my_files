# Лабораторна робота №18: Масиви та функції

## Тема
Використання методів масивів та створення функцій у JavaScript.

## Виконані завдання
1. **Методи масивів:**
   - `map()`: для створення нового масиву з подвоєними значеннями.
   - `filter()`: для відбору парних чисел.
   - `find()`: для пошуку першого елемента, що відповідає умові.
   - `forEach()`: для ітерації по елементах масиву.
2. **Типи функцій:**
   - Реалізовано стандартні функції з ключовим словом `function`.
   - Написано компактні стрілкові функції `=>`.
   - Використано параметри за замовчуванням.
3. **Об'єкти та методи:**
   - Створено об'єкт `student` із методом `showInfo`, який використовує `this` для звернення до властивостей.

## Висновок
У ході роботи було опановано інструменти для роботи з колекціями даних та функціонального програмування в JS. Використання методів масивів дозволяє писати чистіший код, а функції допомагають уникати дублювання логіки.
