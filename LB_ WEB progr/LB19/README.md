# Лабораторна робота №19: Об'єкти в JavaScript

## Виконані завдання:
1. **Об'єкт Teacher:** Створено об'єкт викладача, продемонстровано зміну, додавання та видалення властивостей.
2. **Метод об'єкта:** Додано метод `introduce`, який використовує `this` для доступу до даних об'єкта.
3. **Копіювання та вкладеність:** Створено незалежну копію об'єкта через spread-оператор та додано вкладений об'єкт `car`.
4. **Прототипне наслідування:** Об'єкт `student` (Андрій) успадковує властивості від базового об'єкта `person`.
