# Лабораторна робота №20: Об'єктно-орієнтоване програмування та класи в JavaScript

## Виконані завдання:
1. **Клас Person та об'єкти:** Створено базовий клас із конструктором для ініціалізації імені та віку. Створено кілька екземплярів, зокрема об'єкт для студента Андрія.
2. **Методи та this:** Додано метод introduce, який використовує this для доступу до властивостей об'єкта та виведення привітання в консоль.
3. **Наслідування (Student):** Реалізовано клас Student, що наслідує Person за допомогою extends. Використано super() для передачі даних у батьківський конструктор та додано властивість group (ПЗ-11-11).
4. **Інкапсуляція:** Створено клас BankAccount із приватною властивістю #balance. Продемонстровано неможливість прямого доступу до балансу ззовні класу.
5. **Поліморфізм:** Створено базовий клас Animal та дочірні класи Dog і Cat з різною реалізацією методу speak().



## Як перевірити:
1. Відкрийте файл index.html у браузері.
2. Перейдіть до консолі розробника (F12 -> Console).
3. Перегляньте результати виконання методів для кожного класу, включаючи персоналізовані дані студента групи ПЗ-11-11.
