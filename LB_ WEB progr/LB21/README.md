# Лабораторна робота №21: Деструктуризація та оператори spread/rest

## Виконані завдання:
1. **Деструктуризація масивів:** Реалізовано розпакування списку країн у змінні, використання оператора `rest` для збору цін та встановлення значень за замовчуванням.
2. **Деструктуризація об'єктів:** Створено профіль розробника (Андрій, 39 років) з перейменуванням властивостей та отримання даних із вкладеного об'єкта конфігурації сервера.
3. **Оператор spread:** Продемонстровано об'єднання масивів технологічного стеку та створення копії об'єкта автомобіля (Renault Megane 3) з додаванням нових характеристик.
4. **Оператор rest:** Написано функцію для обчислення суми довільної кількості аргументів та реалізовано відділення ID замовлення від його основних метаданих.
5. **Практичне завдання:** Створено систему управління списком співробітників, де поєднано деструктуризацію в методах масивів та оператор spread для розширення штату.



## Як перевірити:
1. Відкрийте файл index.html у браузері.
2. Перейдіть до консолі розробника (F12 -> Console).
3. Перегляньте результати виконання функцій та виводи об'єктів, включаючи дані студента групи ПЗ-11-11.
