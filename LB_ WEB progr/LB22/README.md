# Лабораторна робота №22: Колбеки та контекст this

## Виконані завдання:
1. **Колбеки:** Створено функцію `processNumber`, яка виконує операції над числами через передані функції-аргументи (множення, піднесення до степеня).
2. **Стрілкові функції:** Реалізовано стрілковий синтаксис та продемонстровано відмінність у поведінці `this` (стрілкові функції не мають власного контексту).
3. **Ключове слово this:** Створено об'єкт студента (Андрій, 39 років) та показано приклад втрати контексту при копіюванні методу об'єкта у змінну.
4. **Методи call, apply, bind:** Використано методи примусової прив'язки контексту до різних об'єктів та створення нових функцій із зафіксованими параметрами.

## Як перевірити:
1. Відкрийте файл **index.html** у браузері.
2. Відкрийте консоль розробника одним із способів:
   - Натисніть праву кнопку миші та оберіть **"Переглянути код елемента"**, потім перейдіть у вкладку **Console**.
   - Натисніть клавіші **Ctrl + Shift + J** (у Chrome/Edge).
   - Натисніть **F12** (якщо клавіша активна).
3. Перегляньте результати роботи для студента групи
