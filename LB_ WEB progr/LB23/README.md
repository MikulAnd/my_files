# Лабораторна робота №23: Перебираючі методи масиву

## Виконані завдання:
1. **forEach:** Реалізовано перебір масиву з виведенням кожного елемента в консоль для аналізу вхідних даних.
2. **map та filter:** Виконано трансформацію значень (подвоєння) та фільтрацію елементів за заданою числовою умовою (більше 39).
3. **find та findIndex:** Реалізовано логіку пошуку першого відповідного елемента та визначення його індексу в структурі масиву.
4. **reduce:** Застосовано метод для агрегації даних та обчислення загальної суми всіх числових елементів.
5. **some, every, includes:** Виконано перевірку масивів на відповідність логічним критеріям та наявність конкретних значень.
6. **splice, slice, sort, reverse:** Продемонстровано методи модифікації складу масиву, виділення фрагментів та керування порядком сортування.
7. **flatMap:** Використано комбінований метод для одночасної трансформації та розгортання вкладених структур даних.



## Як перевірити:
1. Відкрийте файл index.html у браузері.
2. Відкрийте консоль розробника (Права кнопка миші -> Переглянути код -> вкладка Console).
3. Ознайомтеся з результатами виконання операцій над масивами.
