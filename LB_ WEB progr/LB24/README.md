# Лабораторна робота №24: Прототипи та спадкування

## Виконані завдання:
1. **Робота з __proto__:** Створено об'єкти `person` та `student`, реалізовано прототипний зв'язок між ними для успадкування методів.
2. **Методи Object:** Продемонстровано використання `Object.setPrototypeOf` та `Object.getPrototypeOf` для динамічного керування прототипами об'єктів.
3. **Функції-конструктори:** Реалізовано створення екземплярів через `new` з винесенням методів у властивість `prototype` для оптимізації пам'яті.
4. **Затінення методів:** Виконано перевизначення методу прототипу власним методом об'єкта (на прикладі об'єкта `car`).
5. **Прототипне спадкування:** Створено ієрархію конструкторів (`User` та `Admin`) із використанням `Object.create` для передачі методів у ланцюгу спадкування.

## Як перевірити:
1. Відкрийте файл **index.html** у браузері.
2. Відкрийте консоль розробника: права кнопка миші -> **Переглянути код елемента** (Inspect) -> вкладка **Console**.
3. Ознайомтеся з результатами виведення об'єктів та роботи успадкованих методів.
