# Лабораторна робота №25: Процес обробки подій

## Виконані завдання:
1. **Фаза захоплення:** Налаштовано обробники для відстеження події від кореня до цільового елемента.
2. **Цільова фаза:** Реалізовано обробник безпосередньо на кнопці для фіксації події.
3. **Фаза всплиття:** Простежено підйом події від кнопки до батьківських контейнерів.
4. **Керування подіями:** Застосовано методи `stopPropagation` та `stopImmediatePropagation` для зупинки поширення подій.
5. **Делегування:** Реалізовано один обробник для всього списку меню через батьківський елемент `ul`.

## Як перевірити:
1. Відкрийте файл **index.html**.
2. Відкрийте консоль (F12 або Права кнопка миші -> Переглянути код -> Console).
3. Протестуйте кліки по блоках та кнопці для перегляду черговості фаз.
