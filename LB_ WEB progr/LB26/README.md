# Лабораторна робота №26: Маніпулювання HTML-елементами через DOM API

## Виконані завдання:
1. **Отримання елементів:** Використано методи `getElementById` та `querySelector` для доступу до текстового поля, кнопки та списку завдань.
2. **Створення елементів:** Реалізовано функціонал динамічного створення нових елементів списку (`li`) та кнопок керування через `createElement`.
3. **Зміна вмісту та стилів:** Налаштовано передачу тексту з інпуту до нових елементів та перемикання класів для візуального відображення виконаних завдань.
4. **Видалення вузлів:** Додано можливість повного видалення елементів із дерева DOM при натисканні на відповідну кнопку.
5. **Взаємодія з користувачем:** Реалізовано обробку подій кліку та очищення форми після успішного додавання запису.

## Як перевірити:
1. Відкрийте файл **index.html** у браузері.
2. Введіть текст у поле "Нове завдання" та натисніть "Додати".
3. Натисніть "Виконано", щоб закреслити текст, або "Видалити", щоб прибрати елемент зі сторінки.
