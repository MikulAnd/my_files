# Лабораторна робота №28: Web Storage (LocalStorage та SessionStorage)

## Виконані завдання:
1. **LocalStorage:** Реалізовано збереження налаштувань кольорової теми сайту. Вибір користувача зберігається між сесіями браузера.
2. **SessionStorage:** Налаштовано тимчасове збереження даних форми. Введені дані відновлюються після оновлення сторінки, але видаляються після закриття вкладки.
3. **Робота з JSON:** Використано методи `JSON.stringify()` та `JSON.parse()` для серіалізації складних об'єктів (даних форми) перед записом у сховище.
4. **Методи Web Storage:** Продемонстровано практичне застосування методів `setItem()` та `getItem()` для керування станом додатка.

## Як перевірити:
1. Відкрийте файл **index.html**.
2. **Перевірка LocalStorage:** Натисніть кнопку зміни теми та оновіть сторінку. Обрана тема залишиться активною.
3. **Перевірка SessionStorage:** Введіть текст у поля форми та оновіть сторінку — текст залишиться. Закрийте вкладку і відкрийте знову — поля будуть порожніми.
4. Перевірте вкладку **Application -> Storage** у консолі розробника, щоб побачити збережені ключі та значення.
