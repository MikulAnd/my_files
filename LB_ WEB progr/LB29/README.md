# Лабораторна робота: Таймери та об'єкт Date

## Виконані завдання:
1. **Поточний час:** Реалізовано динамічний годинник, який оновлюється щосекунди за допомогою `setInterval` та об'єкта `Date`.
2. **Зворотний відлік:** Створено функцію таймера, яка відраховує задану кількість секунд та виводить повідомлення після завершення.
3. **Будильник:** Налаштовано перевірку часу в реальному часі. При досягненні заданої години та хвилини спрацьовує сповіщення та змінюється колір фону сторінки.
4. **Маніпуляції з датами:** Використано методи `getFullYear`, `getMonth` та `getDate` для виводу поточної дати у зручному форматі.
5. **Керування таймерами:** Застосовано `clearInterval` для зупинки виконання фонових процесів (таймера та перевірки будильника).

## Як перевірити:
1. Відкрийте файл **index.html**.
2. Перевірте роботу годинника (має йти в реальному часі).
3. Введіть кількість секунд у полі таймера та натисніть "Старт".
4. Вкажіть час для будильника (наприклад, на 1 хвилину пізніше поточного) і дочекайтеся спрацювання.
