# Лабораторна робота 30: Пагінація

## Виконані завдання:
1. **Клієнтська пагінація:** Реалізовано поділ масиву з 50 елементів на сторінки. Навігація здійснюється кнопками "Попередня" та "Наступна" з оновленням індикатора сторінки.
2. **Серверна пагінація:** Використано Fetch API для отримання даних з зовнішнього ресурсу (JSONPlaceholder) частинами за допомогою параметрів `_page` та `_limit`.
3. **Інфінітна прокрутка:** Налаштовано автоматичне завантаження нових даних при досягненні користувачем низу сторінки.
4. **Динамічне оновлення DOM:** Нові елементи створюються через `createElement` та додаються до списку без перезавантаження сторінки.
5. **Індикація завантаження:** Додано лоадер, який відображається під час очікування відповіді від сервера.

## Як перевірити:
1. Відкрийте файл **index.html**.
2. У першому розділі протестуйте перехід між сторінками (по 5 елементів на сторінку).
3. Прокрутіть сторінку до самого низу — ви побачите, як автоматично підвантажуються пости з сервера.
