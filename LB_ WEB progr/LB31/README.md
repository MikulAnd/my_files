# Лабораторна робота №31: Проміси та синтаксис async/await

## Виконані завдання:
1. **Створення промісів:** Реалізовано об'єкт `Promise` для імітації асинхронної операції із затримкою у 3 секунди. Результат обробляється за допомогою методу `.then()`.
2. **Обробка винятків:** Створено проміс із логікою випадкового успіху або помилки. Для перехоплення та виведення помилок використано метод `.catch()`.
3. **Паралельні запити:** Реалізовано функцію для одночасного виконання кількох мережевих запитів до API через `Promise.all`, що дозволяє отримувати дані швидше.
4. **Синтаксис Async/Await:** Застосовано сучасний підхід до написання асинхронного коду, який робить логіку запитів до сервера більш зрозумілою та схожою на синхронну.
5. **Робота з API та DOM:** Налаштовано отримання реальних даних із зовнішнього ресурсу та їх динамічне відображення на сторінці через створення нових HTML-елементів.
6. **Конструкція Try/Catch:** Додано надійну обробку помилок усередині асинхронних функцій для запобігання збоїв у роботі додатка.

## Як перевірити:
1. Відкрийте файл **index.html** у будь-якому сучасному браузері.
2. Відкрийте консоль розробника (клавіша F12) для відстеження статусів виконання промісів.
3. Дочекайтеся завершення асинхронних операцій: на сторінці має з'явитися блок із текстом поста, а в консолі — результати паралельних запитів.
