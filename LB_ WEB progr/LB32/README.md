# Лабораторна робота №32: Модулі в JavaScript

## Виконані завдання:
1. **Створення модулів:** Створено окремий файл `utils.js` для винесення допоміжної логіки.
2. **Export:** Реалізовано іменований експорт функцій (`add`, `multiply`) та експорт за замовчуванням.
3. **Import:** Налаштовано імпорт функцій у головний файл `script.js`.
4. **Модульна структура:** Використано атрибут `type="module"` для коректної роботи браузера з декількома файлами.
5. **Асинхронність у модулях:** Продемонстровано поєднання роботи модулів із синтаксисом `async/await`.

## Як перевірити:
1. Відкрийте файл **index.html** (бажано через Live Server у VS Code).
2. Відкрийте консоль розробника.
3. Перевірте вивід результатів математичних операцій та даних з API.
