# Підсумкова робота: Менеджер текстових нотаток (Notes App)

Це фінальний проєкт, що об'єднує знання HTML, CSS та JavaScript. Додаток дозволяє створювати, редагувати та видаляти нотатки, забезпечуючи їх надійне зберігання в пам'яті браузера.

## 📋 Основні можливості
- **Створення нотаток:** швидке додавання нових текстових записів.
- **Редагування:** можливість змінити текст уже існуючої нотатки.
- **Видалення:** очищення списку від непотрібних записів.
- **LocalStorage:** усі дані зберігаються локально, тому вони не зникають після оновлення сторінки.

## 🛠 Технологічний стек
- **HTML5:** структура додатка.
- **CSS3:** адаптивний дизайн та стилізація (вигляд стікерів).
- **JavaScript (ES6+):**
  - Використання **класів (Classes)** для моделювання об'єктів нотаток.
  - Робота з **DOM API** для динамічного оновлення інтерфейсу.
  - Обробка подій через **EventListener**.

## 🚀 Як переглянути проект
Проєкт розгорнуто на **Netlify**. Ви можете скористатися живим посиланням у налаштуваннях репозиторію або відкрити `index.html` локально.
