# Підсумкова робота: Менеджер текстових нотаток (Notes App)

Це фінальний проєкт, що об'єднує знання HTML, CSS та JavaScript. Додаток дозволяє створювати, редагувати та видаляти нотатки, забезпечуючи їх надійне зберігання в пам'яті браузера, а також демонструє роботу з зовнішніми АРІ.

## 📋 Основні можливості
- **Додавання нотаток:** створення записів із заголовком та основним текстом.
- **Редагування:** можливість змінити зміст існуючої нотатки.
- **Видалення:** швидке очищення списку.
- **LocalStorage:** дані автоматично зберігаються в браузері.
- **Інтеграція з API:** асинхронне отримання поради дня з перекладом на українську мову.

## 🛠 Технологічний стек
- **HTML5 & CSS3:** структура та адаптивна стилізація.
- **JavaScript (ES6+):** класи (ООП), DOM API, Async/Await (Fetch), Local Storage.
