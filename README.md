# Підсумкова лабораторна робота: Додаток "Мій розклад"

Цей проект є фінальною роботою курсу JavaScript, що демонструє навички роботи з DOM, об'єктно-орієнтованим програмуванням та збереженням даних у браузері.

## 🚀 Функціонал:
- **Додавання завдань:** створення нових записів у реальному часі.
- **Видалення:** можливість очистити список від непотрібних справ.
- **Збереження (LocalStorage):** ваші плани не зникають після оновлення сторінки або закриття браузера.
- **Адаптивний дизайн:** коректне відображення на мобільних пристроях та ПК.

## 🛠 Технології:
- **HTML5 & CSS3:** структура та стилізація інтерфейсу.
- **JavaScript (ES6+):** - Використання **класів** для моделювання даних (ООП).
  - Маніпуляції з **DOM** для динамічного оновлення списку.
  - Робота з **Local Storage** для стабільного зберігання даних.

## 📦 Як запустити локально:
1. Клонуйте репозиторій.
2. Відкрийте `index.html` у будь-якому сучасному браузері.
