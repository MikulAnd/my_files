# Підсумкова робота: Менеджер текстових нотаток (Notes App)

Це фінальний проєкт, що об'єднує знання HTML, CSS та JavaScript. Додаток дозволяє створювати, редагувати та видаляти нотатки, забезпечуючи їх надійне зберігання в пам'яті браузера, а також демонструє роботу з зовнішніми АРІ.

## 📋 Основні можливості
- **Додавання нотаток:** створення нових записів із заголовком та основним текстом.
- **Редагування:** можливість змінити зміст уже існуючої нотатки без її видалення.
- **Видалення:** швидке очищення списку від непотрібних справ.
- **LocalStorage:** усі дані автоматично зберігаються в браузері, тому вони доступні навіть після перезавантаження сторінки.
- **Інтеграція з API:** динамічне підвантаження випадкових порад з ресурсу *Advice Slip API* для покращення досвіду користувача.

## 🛠 Технологічний стек
- **HTML5 & CSS3:** структура додатка та адаптивний дизайн (вигляд сучасних стікерів).
- **JavaScript (ES6+):**
  - Використання **класів (Classes)** для моделювання об'єктів нотаток (ООП).
  - Робота з **DOM API** для динамічного оновлення інтерфейсу без перезавантаження.
  - Асинхронність через **Async/Await** та **Fetch API** для отримання даних з мережі.
  - Робота з **Local Storage** для стабільного зберігання даних користувача.
