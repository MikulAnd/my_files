# Підсумкова робота: Менеджер текстових нотаток (Notes App)

Це фінальний проєкт, що об'єднує знання HTML, CSS та JavaScript. Додаток дозволяє керувати нотатками та демонструє роботу з асинхронними запитами.

## 📋 Основні можливості
- **Створення та заголовок:** кожна нотатка має назву та зміст.
- **Пошук:** "жива" фільтрація нотаток за ключовими словами в реальному часі.
- **Редагування:** можливість оновити текст існуючої нотатки.
- **LocalStorage:** дані зберігаються в браузері та не зникають після оновлення.
- **Інтеграція з API:** отримання поради дня через *Advice Slip API* з автоматичним перекладом на українську мову.

## 🛠 Технологічний стек
- **HTML5 & CSS3:** структура та стилізація.
- **JavaScript (ES6+):** класи (ООП), DOM API, Async/Await (Fetch API), Local Storage.
